## What is GitHub Pages?

GitHub Pages is a static site hosting service that takes HTML, CSS, and JavaScript files directly from a repository on GitHub, optionally runs the files through a build process, and publishes a website.

## Why I'm Using GitHub Pages

- **Free Hosting**: GitHub Pages offers free hosting for personal, organization, and project pages
- **Easy to Use**: No need for complex server setups or configurations
- **Version Control**: All my content is version controlled with Git
- **Jekyll Integration**: Built-in support for Jekyll, a static site generator
